# DevPortfolio
![CI](https://github.com/Cr3zy-Hyena/devportfolio/actions/workflows/ci.yml/badge.svg)
![Python](https://img.shields.io/badge/python-3.11-blue)
![Docker](https://img.shields.io/badge/docker-ready-blue)

Application web de portfolio pour developpeurs.
Stack : Flask, PostgreSQL, Docker, GitHub Actions, Nginx.

## Lancement rapide
```bash
git clone https://github.com/Cr3zy-Hyena/devportfolio.git
cd devportfolio
cp .env.example .env
docker compose up -d
```
## Stack technique
| Composant | Technologie |
| ----------- | -------------|
| Backend | Python 3.11 + Flask |
| BDD     | PostgreSQL 15 |
| Conteneurs | Docker + Compose |
| CI/CD | Github Actions |
